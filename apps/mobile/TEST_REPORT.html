📝 Замечания

⚠️ Jest Watch Plugins Warnings
Есть warnings про watchPlugins из jest-watch-typeahead. Это не влияет на результаты тестов и является особенностью jest-expo. Не требует фиксации для локального запуска.
✅ Использование pnpm
Проект использует монорепозиторий с pnpm. Правильный способ запуска:
pnpm --filter mobile test # или cd apps/mobile && npx jest --no-coverage

🚀 Инструкция по запуску

Базовый запуск

pnpm --filter mobile test

С отчётом в JSON

cd apps/mobile npx jest --json --outputFile=test-report.json

С code coverage

npx jest --coverage

Watch mode (для разработки)

npx jest --watch
